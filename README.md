# Visual Adder Simulator (可視化機能付き 加算器シミュレータ)

Pythonの標準GUIライブラリ `tkinter` を使用して作られた、半加算器（Half Adder）および全加算器（Full Adder）の論理回路シミュレータです。
<img width="874" height="720" alt="スクリーンショット 2026-05-31 090734" src="https://github.com/user-attachments/assets/ce155acc-02f5-44ac-ab28-6b939ba23314" />

入力信号の切り替え（0/1）に応じて、**回路上のどこに電気が流れているか（通電状態）がリアルタイムに色で判別できる**ため、論理回路の仕組みを視覚的に楽しく学ぶことができます。

## ✨ 特徴

- **リアルタイム可視化**: 入力を切り替えると、信号が `1` のルート（通電）は**赤色**、`0` のルート（非通電）は**黒色**に即座に変化します。
- **2つのモードを搭載**: 
  - **半加算器モード**: 2入力（A, B）の基本的な加算を表示。
  - **全加算器モード**: 3入力（A, B, Cin）に対応し、内部で2つの半加算器と1つのORゲートがどう接続されているかを可視化。
- **スマートUI**: 半加算器モードの時は、不要な入力（Cin）が自動的にグレーアウトされ、誤操作を防ぎます。
- **外部ライブラリ不要**: Pythonの標準ライブラリのみで記述されているため、追加のインストールなしで即座に実行可能です。

## 🚀 使い方

### 1. 実行環境
- Python 3.x がインストールされていること。

### 2. 起動方法
リポジトリをクローンするか、ソースコードを `visual_adder.py` として保存し、以下のコマンドを実行します。

```bash
python visual_adder.py
3. 操作方法画面上部の「回路選択」から、半加算器または全加算器を選択します。「信号入力」のチェックボックスをクリックして、入力を 0（オフ）または 1（オン）に切り替えます。下部のキャンバスにリアルタイムで計算結果（Sum / Carry）と、電気の流れが描画されます。🛠 内部ロジックの仕組み本アプリケーションは、実際のデジタル電子回路の構成を忠実に再現しています。半加算器 (Half Adder)$Sum = A \oplus B$ （XORゲート）$Carry = A \cdot B$ （ANDゲート）全加算器 (Full Adder)1つ目の半加算器で $A$ と $B$ を計算。その結果の $Sum$ と、下位からの桁上げ（$Cin$）を2つ目の半加算器に入力。各半加算器から出力された桁上げ信号を ORゲート で結合し、最終的な $Cout$ を出力。📦 ファイル構成.
